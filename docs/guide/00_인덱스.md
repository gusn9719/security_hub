## 프로젝트 한 줄 정의

의심 문자/URL을 다신호 스코어링 파이프라인으로 분석해 `안전/의심/위험`을
판정하고, 애매하면 가상 샌드박스(직접 탐방 또는 AI 자동 테스트)로 사용자가
확인하게 하는 앱. **핵심 차별점**: 사용자가 샌드박스에서 투표한 결과가
`url_votes`에 쌓여 다음 분석의 휴리스틱 시그널로 즉시 반영되는 피드백
순환 구조 — "그냥 차단 도구가 아니라 학습형 위협 인텔리전스".

```
신규 위협 URL → 휴리스틱 30~69점(SUSPICIOUS) → 사용자가 7-A에서 직접 확인
→ danger 투표 → url_votes 누적 → 다음 분석 시 prior_danger_vote_* 가산
→ 누적되면 DANGER로 승격
```
