# 사용자 흐름

## 전체 흐름 (문자 수신 → 분석 → 가상 환경)

![E2E 흐름](E2E.png)

문자를 받으면 복사해서 앱에 붙여넣고 분석하기를 누릅니다. 결과를 확인한 뒤 판정이 애매하면(의심) 가상 환경을 실행해서 직접 조작해보고 앱을 종료하는 흐름입니다. 실제로는 문자 복사·붙여넣기 외에 공유하기, 클립보드 자동 감지, 문자함 불러오기, SMS 수신 알림으로도 같은 분석 파이프라인에 진입할 수 있습니다.

## 화면 구성

![메뉴 구성도](메뉴구성도.png)

홈 화면에서 문자를 입력하면 안전/의심/위험 3갈래로 결과 화면이 나뉩니다. 위험이면 발신번호 차단으로, 의심이면 가상 샌드박스(7-A 직접 탐방 또는 7-B AI 자동 테스트) 선택으로, 안전이면 URL을 바로 열 수 있는 화면으로 이어집니다.
